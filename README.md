# Aviation Phonetic Alphabet Converter — Free Embeddable Widget

A free, zero-dependency **NATO/ICAO phonetic alphabet converter** widget for aviation websites, flight schools and pilot blogs. Type a word, callsign or aircraft registration and it spells it out in the aviation phonetic alphabet (Alpha, Bravo, Charlie…).

**[▶ Live demo](https://aviationshop.github.io/aviation-phonetic-widget/)**

## Embed it (free)
Paste where you want the widget to appear:
```html
<div id="avshop-phonetic"></div>
<script src="https://cdn.jsdelivr.net/gh/AviationShop/aviation-phonetic-widget@main/widget.js"></script>
```
No dependencies, ~3 KB, responsive. Free to use — please keep the small attribution link.
